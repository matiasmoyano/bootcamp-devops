Examen Git y Github

Alumn@: Matias Moyano

  1. Que es un sistema de control de versiones?

    Es un sistema que permite monitorear y gestionar los cambios de versiones en un sistema de archivos

  2. Cual es la diferencia entre git y github?

    la diferencia es que github es un servicio donde se pueden guardar repositorios creados
    con git, y git es un software para registrar cambios en archivos que permite a los desarrolladores
    colaborar sobre un mismo proyecto usando lineas de desarrollo distintas gracias a la creacion
    de branches

  3. Cómo creo una rama en git?

    Para crear una rama en git uso el comando git checkout -b nombre_branch

  4. Cómo clono un repositorio?

    Para clonar un repositorio uso el comando git clone url_de_repo
